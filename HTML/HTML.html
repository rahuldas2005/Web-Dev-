WHAT IS HTML?

As, you should know that:

  • HTML is not a programming launguage but it is a Hyper Text Markup Language.
  • It is made up of ELEMENTS and TAGS.
  • It is a text which we used to give instruction to the browser what to display on a web page.

WHAT ARE HTML ELEMENTS?

HTML ELEMENTS means, individual components of an HTML and under each elements there were tags to display their own meaning.

    For Example: HTML TAGS, IMG TAGS etc...

NESTING OF ELEMENTS

It meants, under 1 element there eill be another elements.....

    For Example: We can write BOLD tag under P tag....

If we just used only elements or tags inour code then it will be just look like a google document. So, we need attributes to add style to the elements to look more beautiful.
