So the STRUCTURE OF A WEBSITE is mainly consist of the semantic tags and non-semantic tags. Semantic tags like: HEADER> NAV> MAIN> SECTION> ASIDE> ARTICLE> FOOTER.


WHAT ARE SEMANTIC TAGS?


Tags which is clearly defines the content.
For example: h1, h2, h3, p, hr, br, etc...


WHAT ARE NON-SEMANTIC TAGS?

Tags which is not-clearly defines the content.
For exampl: div, span, etc...


DIV and SPAN are the BLOCK LEVEL and INLINE LEVEL of tags.
