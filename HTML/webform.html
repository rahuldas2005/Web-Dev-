FORM

Form is mainly used to collect data from the consumers. the form attributes are of two types which are discused below:

  • To collect data.
  • To send data to other server or our main server.

There are various way to get or collect data from the users, following are the some ways of it..

  1. INPUT TYPE = "text"
  2. INPUT TYPE = "password"
  3. INPUT TYPE = "email"
  4. INPUT TYPE = "radio"
  5. INPUT TYPE = "submit"

These are few steps by which we can get or collect data from the users.
