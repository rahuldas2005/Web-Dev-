WHAT ARE ATTRIBUTES?

At every elements we can add some additional properties like,

    For Example: On IMAGE tag, we can see the attributes like, "location.jpg" "width", etc...

WHAT ARE BOLLEN ATTRIBUTES?

These are some useless attributes that are not required.


WHAT ARE GENERAL PURPOSE ATTRIBUTES?

There are 2 types os attributes; a) ID Attribute and b) CLASS Attribute.

  1. ID: This is unique element i.e, a unique value can be used and the unique id will never be repeated for other.
      2. For Example: <.p id = "First"> <./p>
      On CSS, we will write; #First {color:red}

  2. CLASS: This is a non-unique i.e, we can give a unique to more than one element which is not possible by ID attribute.
      3. For Example: <.p class = "Second"> <./p>
      For Example: <.p class = "Second"> <./p>
